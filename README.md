# Drake

Model-Based Design and Verification for Robotics.

Please see the [Drake Documentation](https://drake.mit.edu) for more
information.

## About this fork

This fork provides additional python bindings for certain useful calculations,
mainly for elements of the (depreciated) RigidBodyTree framework.

## Workflow

Build and test: `bazel test //...`

Install to `[install_dir]`: `bazel run install [install_dir]`
